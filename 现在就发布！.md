## 🎯 方式1: 使用半自动脚本（推荐）⭐⭐⭐⭐⭐

### 第1步：打开终端运行脚本

```bash
cd /Users/kanelli/workspace/xhs-auto-publisher
source venv/bin/activate
python interactive_publish.py
```

### 第2步：按回车启动浏览器

终端会显示操作说明，按回车后浏览器会自动打开

### 第3步：在浏览器中操作（您需要做的）

1. **点击"上传图文"标签**
   - 在页面顶部，有三个标签
   - 点击中间的"上传图文"

2. **上传3张图片**
   - 点击上传按钮
   - 选择这3张图片（按顺序）:
     * image1_title.jpg （标题图）
     * image2_list.jpg （列表图）
     * image3_guide.jpg （指南图）
   - 图片位置: `output/contract_xhs/`

3. **等待图片上传完成**
   - 看到3张图片都显示在页面上
   - 进入编辑界面（有标题和内容输入框）

### 第4步：回到终端按回车

完成图片上传后，回到终端，按回车键

### 第5步：脚本自动完成剩余工作

脚本会自动：
- ✅ 填写标题
- ✅ 填写内容和标签
- ✅ 点击发布按钮

### 第6步：确认发布

- 输入 `yes` 并回车：自动点击发布
- 或直接回车：您手动点击发布按钮

